一、本周核心工作进展

本周核心目标为打破 ISAR 图像中「类别 - 分辨率」虚假关联，让模型聚焦分辨率本身的物理特征（散射点密度、纹理精细度、像素梯度分布等），同时实现分辨率插值的物理级模拟增强，最终完成所有

方案的代码落地、问题修复与训练流程启动，具体工作如下：

完成分辨率 - 类别特征解耦全流程代码落地，结合双分支解耦网络、互信息损失约束、分辨率对比学习等核心策略，编写适配 ISAR 数据的数据集类、模型结构、损失函数与训练逻辑；

实现基于 ISAR 成像物理的分辨率插值增强，编写分辨率退化 / 插值模拟函数，支持跨分辨率数据生成，打破类别与分辨率的强耦合关系；

完成全量代码调试与环境适配，解决编码异常、模块导入、路径配置、数据集加载等系列问题，确保解耦训练流程无报错；

完成原始 ISAR 数据的划分与预处理，构建均衡的「类别 - 分辨率」训练集 / 验证集 / 测试集，匹配解耦训练数据要求；

成功启动两阶段解耦训练流程（对比预训练 + 解耦微调），当前模型在 GPU 环境下正常运行，无代码与逻辑层面报错。

二、核心工作详情：分辨率解耦与插值增强落地实现

（一）数据层面：实现物理级分辨率插值增强，构建无偏训练集

针对训练集「类别 - 分辨率」强耦合导致的虚假关联问题，从数据分布平衡与分辨率插值增强两方面入手，落地无偏训练集构建方案，核心代码实现于fb/dataset_resolution.py：

类别 - 分辨率分布平衡：编写数据重采样逻辑，统计所有「类别 - 分辨率」组合的样本数量，对低样本组合进行过采样、高样本组合进行欠采样，确保各组合样本分布均衡，避免模型学习类别与分辨率

的固定绑定关系；

核心：物理级分辨率插值 / 退化模拟：基于 ISAR 成像物理模型（带宽 / 积累角影响分辨率），编写simulate_isar_res_degradation函数，实现高 / 低分辨率图像的跨分辨率插值生成，模拟真实的分辨率物理特征变化：

低分辨率插值（上采样）：对低分辨率图像采用双三次 / 兰索斯插值放大，结合拉普拉斯算子增强高频细节，模拟超分辨率效果；

高分辨率退化（下采样）：对高分辨率图像先进行高斯模糊（模拟带宽降低导致的散射点扩散），再进行区域插值下采样，匹配目标分辨率的物理特征；

插值缩放因子由原始分辨率与目标分辨率的比值动态计算，确保插值后的图像符合 ISAR 分辨率的物理规律；

跨分辨率数据增强：构建全分辨率插值列表，对每个样本生成多组不同分辨率的插值版本，人工构建「同一类别、不同分辨率」的样本集，强制模型学习同一目标的分辨率变化特征，而非类别特征；

混合增强约束：在分辨率插值基础上，叠加随机旋转、亮度抖动、高斯模糊、随机裁剪等操作，避免模型依赖「类别 + 角度 + 分辨率」的组合特征，进一步解耦类别与分辨率关联。

（二）模型层面：落地双分支解耦网络，强制学习分辨率专属特征

针对特征学习目标错位问题，设计共享骨干 + 双分支输出的解耦网络结构，结合注意力机制与解耦损失约束，让模型分别学习类别特征与分辨率特征，核心代码实现于models/decoupled_head.py与fb/train_resolution.py：

双分支解耦网络构建：基于 LWGANet 骨干网络，搭建解耦网络DecoupledISARNet，共享骨干提取图像基础特征（散射点、纹理、梯度），再分两个独立分支输出：

类别分支：负责目标类别分类，学习类别专属特征；

分辨率分支：负责分辨率估计，学习分辨率专属特征；
